A ideia é primerio selecionamos o valor i do primerio for como
candidato a valor minimo, pois primeiro temos que encontra o valor
minimo para fazer a troca com valor i do primerio for
ou seja o primeiro for fix uma posição do array e o segundo verifica se existe
um numero menor que aquele na demais posições do array,
caso ache então faz a troca de posições até que todo os valores estejam ordenado
selection_sort(A[], tamanho){
    
    for i <- 0 ate para tamanho-1{

        // candidato a valor minimo do array
        valor_minimo <- i

        // seleciona o menor valor do array    
        for(j <- i + 1 para tamanho){

            if(A[j] < A[valor_minimo]) entao {

                valor_minimo <- j
            }

        }

            // troca de posição
            temp <- A[i]

            A[i] = A[valor_minimo]

            A[valor_minimo] = temp

    }
}
