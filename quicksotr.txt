Seleciona um pivor(neste caso o ultimo elemento do array) e todos os elementos menor que o pivor vai para a 
esquerda do array, e os que forem maior vai para direita do array
tudo em apenas um array, ou seja dividindo em 2 sub-array(parte direita e esquerda do mesmo array) para que a ordenação
seja feita no lado direito do array e no lado esquerdo do array
na recursão é ordenado o lado direito do array e o lado esquerdo.

inteiro particao_array(inteiro A[], inteiro comeco , inteiro fim){

    inteiro pivo <- A[fim]

    inteiro pindex <- comeco

    para i<-comeco ate fim-1{

        se(A[i] <= pivo) entao {

            troca(A[i], pindex)

            pindex <- pindex + 1

        }

    }

    troca(A[pindex], A[fim]);

    retorna pindex

}

quicksort(inteiro A[], inteiro comeco, inteiro fim){

    if(comeco >= fim) then {

        return;

    }

        inteiro pindex <- particao_array(A, comeco, fim)

        quicksort(A,comeco, pindex - 1)

        quicksort(A, pindex+1, fim)


}
